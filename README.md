# Health and Wellness Lifestyle Solver

A comprehensive SwiftUI-based health and wellness lifestyle optimization application that uses parameter sweeping and optimization algorithms to create personalized meal plans, daily schedules, and wellness plans based on USDA nutrient guidelines, blood test results, personal preferences, and journal analysis. The app integrates with your calendar to schedule your day and provides planning sessions for day start/end, week start/end, and month start/end periods.

## Usage

### Basic Usage

```swift
import DietSolver
import SwiftUI

@main
struct MyApp: App {
    var body: some Scene {
        WindowGroup {
            ContentView()
        }
    }
}
```

### Programmatic Usage

```swift
import DietSolver

// Create health data
let healthData = HealthData(
    glucose: 95.0,
    hemoglobin: 14.5,
    cholesterol: 180.0,
    bloodPressure: HealthData.BloodPressure(systolic: 120, diastolic: 80),
    age: 35,
    gender: .male,
    weight: 75.0,
    height: 175.0,
    activityLevel: .moderate
)

// Solve diet
let solver = DietSolver()
let season = Season.spring
let dietPlan = solver.solve(healthData: healthData, season: season)

// Generate outputs
let recipe = RecipeGenerator.generateRecipe(for: dietPlan.meals[0])
let song = SongGenerator.generateSong(for: dietPlan)
let nutritionFacts = NutritionFactsGenerator.generateNutritionFacts(
    for: dietPlan,
    requirements: healthData.adjustedNutrientRequirements()
)
```

## Optimization Algorithm

The solver uses an iterative improvement approach:

1. **Initialization**: Random amounts of available seasonal foods
2. **Scoring**: Calculates fitness based on:
   - Nutrient deficiencies/excesses
   - Taste scores
   - Digestion scores
   - Food variety
3. **Optimization**: Iteratively adjusts food amounts to minimize score
4. **Meal Distribution**: Distributes optimized foods across breakfast, lunch, and dinner

## Medical Test Integration

### Automatic Nutrient Adjustments
The system automatically adjusts diet recommendations based on medical test results:

- **Blood Glucose/HbA1c**: Reduces carbohydrate requirements for diabetes/prediabetes
- **Hemoglobin/Ferritin**: Increases iron requirements for anemia
- **Vitamin Deficiencies**: Boosts specific vitamin requirements (D, B12, etc.)
- **Heavy Metals**: Increases selenium for mercury detox, calcium/iron for lead binding
- **Organ Conditions**: Adjusts nutrients based on organ health (liver, kidney, etc.)
- **Hormonal Imbalances**: Modifies nutrients for hormonal support

### Analysis Features
- **Issue Detection**: Identifies abnormal values and potential health concerns
- **Warning System**: Flags borderline values requiring attention
- **Trend Analysis**: Tracks changes over time (when multiple tests available)
- **Recommendation Engine**: Generates specific dietary, exercise, and lifestyle recommendations

## Customization

You can extend the food database by adding more foods to `FoodDatabase.loadFoods()`:

```swift
foods.append(Food(
    name: "Your Food",
    category: .vegetable,
    nutrientContent: NutrientContent(...),
    properties: FoodProperties(...)
))
```

You can also extend the exercise database by adding activities to `ExerciseDatabase.loadActivities()`:

```swift
activities.append(ExerciseActivity(
    name: "Your Exercise",
    category: .cardio,
    intensity: .moderate,
    caloriesPerMinute: 0.08,
    benefits: ["Benefit 1", "Benefit 2"],
    muscleGroups: ["Muscle Group"]
))
```
